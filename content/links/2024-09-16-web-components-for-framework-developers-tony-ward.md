> Web Components are great for building Design Systems that need to work across multiple frameworks. If you've been working in framework-land for quite some time now, you may find some topics confusing at first.
 
 [Web Components for Framework Developers](https://www.tonyward.dev/articles/web-components-for-framework-devs) by Tony Ward does a great job breaking down some of the unique aspects of Web Components for developers who are used to working with components in frameworks like React and Angular.
